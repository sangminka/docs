## 2. 빠른 시작 (5분 안에 끝내기)

### Step 1. 회원가입

1. {{서비스 URL}}에 접속합니다.
2. 우측 상단 **[시작하기]** 버튼을 클릭합니다.
3. 이메일과 비밀번호를 입력합니다.
   > 비밀번호는 8자 이상, 영문·숫자·특수문자 조합이어야 합니다.
4. **[가입하기]**를 누르면 인증 메일이 발송됩니다.
5. 메일함에서 "{{프로젝트명}} 이메일 인증" 메일을 확인하고 링크를 클릭합니다.

### Step 2. 첫 로그인 & 온보딩

1. 인증 완료 후 자동으로 온보딩 화면이 나타납니다.
2. 닉네임과 관심 분야를 선택합니다.
3. **[시작하기]**를 누르면 대시보드로 이동합니다.

## 4. 계정 관리

### 4.1 비밀번호 변경

1. 우측 상단 프로필 아이콘 → **[설정]**
2. **[보안]** 탭 선택
3. 현재 비밀번호와 새 비밀번호 입력 후 **[변경]**

### 4.2 비밀번호를 잊었을 때

1. 로그인 화면에서 **[비밀번호 찾기]** 클릭
2. 가입한 이메일 입력 → 재설정 링크 수신
3. 메일 내 링크를 통해 새 비밀번호 설정

### 4.3 회원 탈퇴

1. **[설정]** → **[계정]** → **[회원 탈퇴]**
2. 안내 사항을 읽고 본인 비밀번호로 확인
3. 탈퇴 후 {{14일}} 동안 데이터가 보관되며, 그 후 영구 삭제됩니다.

## 5. 자주 묻는 질문 (FAQ)

### Q1. 무료로 사용할 수 있나요?

A. {{포트폴리오 프로젝트로, 시연 기간 동안 무료로 사용 가능합니다.}}

### Q2. 모바일에서도 쓸 수 있나요?

A. 모바일 브라우저에서 동작하지만, **데스크톱 환경(1280×800 이상)에 최적화**되어 있습니다.

### Q3. 데이터는 안전한가요?

A. 비밀번호는 BCrypt로 단방향 암호화되어 저장되며, 모든 통신은 HTTPS를 사용합니다. 자세한 내용은 [개인정보처리방침]({{링크}})을 참고하세요.

### Q4. 오류가 발생했을 때 어떻게 해야 하나요?

A. 페이지를 새로고침해 주세요. 그래도 해결되지 않으면 아래 "문제 해결" 섹션을 확인하거나, 화면 우측 하단 **[문의하기]**로 알려주세요.

### Q5. 데이터를 백업할 수 있나요?

A. **[설정]** → **[데이터]** → **[내 데이터 내보내기]**에서 JSON 형식으로 다운로드할 수 있습니다.

## 6. 문제 해결 (Troubleshooting)

| 증상 | 원인 | 해결 방법 |
|------|------|-----------|
| 로그인이 안 됨 | 이메일 인증 미완료 | 메일함에서 인증 링크 클릭 |
| "잘못된 입력" 에러 | 입력 형식 오류 | 폼 안내 문구를 다시 확인 |
| 페이지가 하얗게 나옴 | 브라우저 캐시 | `Ctrl + Shift + R`로 강력 새로고침 |
| 응답이 느림 | 네트워크 일시 지연 | 30초 후 재시도, 그래도 느리면 문의 |
| 한글 검색이 안 됨 | 입력어 끝 공백 | 검색어 끝 공백 제거 (개선 예정) |
